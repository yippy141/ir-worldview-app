Structural compromises · desktop → mobile

What was traded, and why

Layout

  1. Desktop two-column head dropped to one. The right-hand fact card on desktop becomes three wrap-friendly chips above the chart. Same three facts (baseline · changed most · stayed stable), tighter typography.
  2. Chart label column collapsed into the row. Desktop reserves a 320 px label gutter; mobile inlines lane name + anchor numbers above each bar so the bar gets the full width.
  3. Per-lane meta moved below the bar. Module source and Δ value sit on a single line under the bar instead of beside it.
  4. Layer-relationship section moved below the chart. On desktop it sat beside the CTA row; on mobile it sits between the chart and the CTAs so the AI layer is acknowledged before the user is asked to act.

Density

  1. Page nav becomes a horizontal segmented strip. Six nav items don’t fit in a top bar at 390 px; they scroll horizontally with Profile pinned active. The hamburger covers everything else.
  2. Chart scale strip simplified. Long axis labels (“toward coordination / institutions / restraint”) shorten to “← coord.” / “compet. →” to keep the row uncrowded.
  3. Decimals trimmed in row anchors. “3.8 → 5.6 / 7” instead of two separate “baseline / pressure” lines.

Progressive disclosure

  1. Three appendix-style sections collapsed by default. Evidence log, “Where this result will be wrong,” and Trust & coverage all start closed; tapping rotates the chevron and expands inline. None of them are required for the four above-the-fold questions.
  2. CTAs stack vertically, three deep. Same three calls as desktop; stacked at 44 px tall hit-targets so they read as buttons without dominating. The fourth desktop CTA (“Field Guide on shifts”) is deferred to the appendix area.

What did not change

  1. Same dominant visual. Anchored spread chart, same five lanes, same encoding (baseline as anchor, deviation as bar, stable lanes muted in forest).
  2. Same three chips, same modal language. “Leans” and “leans heavily” carry over; “stable” retained as a third category.
  3. Same noun system. Foundation, Modules, AI Governance, Profile, Atlas, Field Guide — pinned to nav at the top.
