## 1. Theoretical fit checks

### How to read the expected answer patterns
- Likert scale: 1 = strongly disagree, 7 = strongly agree
- SC = security competition, IN = institutions, DF = domestic filters, NI = norms/identity, PE = political economy, RS = restraint, OJ = order vs justice
- Scenario answers: A/B/C per the quiz question order

### Expected answer patterns by profile

| Dimension | Expected score | Rationale |
|---|---|---|
| SC (sc1, sc2, sc3) | 6–7 all three | Uncertainty is permanent; self-help is structural |
| IN (in1 reverse, in2 reverse, in3 agree) | 1–2 on in1/in2; 6–7 on in3 | Institutions reflect power, don't constrain it |
